# TesloShop

1. Clonar el repositorio
2. CLonar el archivo `.env.template` y renombrarlo a`.env`
3. Ejecutar el coamndo `pnpm install` para instalar las dependencias
4. Verificar que el backend esté corriendo el puerto `8000`
5. Ejecutar el comando `pnpm run dev` para iniciar el servidor de desarrollo
